## otter node 4.2.12

容器化 otter node 4.2.12 版本

#### 可用环境变量

```shell
NODE_ID=1  
OTTER_MANAGER_ADDRESS=127.0.0.1:1099 
OTTER_ZOOKEEPER_SESSIONTIMEOUT=60000 
OTTER_COMMUNICATION_POOL_SIZE=10 
LOG_LEVEL=INFO 
LOG_APPENDER=STDOUT 
JAVA_XMS=256M 
JAVA_XMX=512M 
JAVA_XMN=128M
```
